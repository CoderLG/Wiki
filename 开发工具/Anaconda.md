查看创建了那些环境

conda info --env 

升级conda

**conda update conda** 

创建环境

conda create -n env_name python=2.7 

移除环境

conda remove -n your_env_name(虚拟环境名称) --all 



进入环境

conda activate py36

退出环境

conda deactivate  py36

conda安装

**conda install -n your_env_name [package]**
